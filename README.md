Centre a div in the page
==========================
```
<div style="margin: 50 auto;">
```

Insert a horizontal line
=========================

```
<div class="col-md-2"> 
            <hr style="width: 1px; height: 500px; background: black; border: none;" />  
 </div>
 ```
 
 Fix to top
 ===========
 Fix an element to the top of the browser.
 
 ```
 .classfix {
 position:fixed;
 top:0;
 width:100%
 z-index:10;
 }
 ```
